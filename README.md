# mr-renovations-site

Next.js 15 website for M.R. Renovations LLC — residential renovation contractor in Maple Grove, Minnesota.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```

## Tech Stack

- Next.js 15 (App Router) with TypeScript
- Tailwind CSS v4 (CSS-first config)
- DM Sans + Inter via `next/font/google`
